Systemcraft

"To understand the machine is not to master it — it is to begin speaking its language." — Systemcraft, Codex I

Systemcraft is the study of how digital systems are built, communicated, and maintained. It begins with the web — the most immediate and accessible canvas available to any practitioner.

The web is built from three layers. Master these three, and you can build nearly anything a browser can display:

The Three Layers

I. HTML — Structure

HTML (HyperText Markup Language) is the skeleton. It defines what exists on a page: headings, paragraphs, links, images, buttons. Without HTML, there is nothing.

II. CSS — Appearance

CSS (Cascading Style Sheets) is the skin. It defines how things look: color, size, spacing, fonts, layout. The same HTML can look completely different with different CSS.

III. JavaScript — Behavior

JavaScript is the nervous system. It defines how things react: clicks, animations, dynamic content that changes without reloading the page.
