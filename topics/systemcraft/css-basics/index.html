CSS Basics

"HTML gives things form. CSS gives them presence. The same structure, dressed differently, becomes a different world." — Systemcraft, Second Lesson

CSS stands for Cascading Style Sheets. It is the language that controls how HTML looks. Every color, font, size, spacing, border, and layout rule you see on this page — including the parchment panels, the gold borders, and the dark background — comes entirely from CSS.

How CSS Works

CSS is written as rules. Each rule has two parts: a selector (which HTML element to target) and declarations (what to change about it). 

/* This is a CSS rule */
/* "p" is the selector — it targets all <p> tags */
p {
    color: #2c1a0e;        /* text color */
    font-size: 1rem;       /* text size */
    line-height: 1.8;      /* space between lines */
    margin-bottom: 16px;   /* space below each paragraph */
}

How to Connect CSS to HTML

You write CSS in a separate file (usually called style.css), then link it to your HTML file using a <link> tag inside the <head>: 

<head>
    <link rel="stylesheet" href="style.css">
</head>

Once linked, every rule in style.css applies to the HTML. Change the CSS — the page updates instantly on refresh.

The Most Important Properties

/* TEXT */
color: #333;                    /* text color */
font-size: 1rem;                /* 1rem = 16px by default */
font-family: 'Georgia', serif;  /* font choice */
font-weight: bold;              /* bold text */
text-align: center;             /* center text */

/* SPACING */
margin: 20px;                   /* space OUTSIDE the element */
padding: 16px;                  /* space INSIDE the element */
margin-top / margin-bottom / margin-left / margin-right

/* SIZE */
width: 300px;                   /* fixed width */
max-width: 800px;               /* won't grow wider than this */
height: 100px;

/* BACKGROUND & BORDER */
background-color: #f5edd6;
border: 1px solid #c9a84c;
border-radius: 8px;             /* rounded corners */

/* BOX MODEL */
box-sizing: border-box;         /* padding included in width calculation */

CSS Variables

CSS variables let you define a value once and reuse it everywhere. This is how this entire grimoire uses one color scheme — change the variable, and everything updates. 

/* Define variables in :root (applies to the whole page) */
:root {
    --my-color: #c9a84c;
    --my-font: 'Georgia', serif;
}

/* Use them anywhere with var() */
h1 {
    color: var(--my-color);
    font-family: var(--my-font);
}
✦ ✦ ✦

Understanding "Cascading"

The "C" in CSS stands for Cascading — meaning rules flow downward and later rules override earlier ones. More specific selectors win over general ones. 

p { color: black; }             /* all paragraphs: black */
.special { color: red; }       /* paragraphs with class="special": red */
#unique { color: gold; }       /* the element with id="unique": gold */

/* id > class > tag — that's the specificity order */

Practice Exercise

Take your practice.html from the HTML chapter. Now:

  • Create a style.css file in the same folder
  • Link it to your HTML with a <link> tag
  • Change the body background color to something other than white
  • Change the h1 color and font-size
  • Add padding and a border to a paragraph
  • Define at least one CSS variable and use it
Most of what you will learn about CSS comes from trying something, seeing the wrong result, and figuring out why. That process is the lesson. The documentation is just a map.
