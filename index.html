About This Codex

"A grimoire is not read from cover to cover. It is consulted, annotated, and returned to. Its value lies in the returning." — Keeper's Note, Vol. I

This grimoire is a living document — it grows as knowledge is gathered, tested, and refined. It is not a finished product. It is a practice.

Pages are added, revised, and reorganized over time. If a chapter feels incomplete, that is because it is. Return again later.
